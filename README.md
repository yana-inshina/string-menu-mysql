# Консольное меню: строки + MySQL + Excel

## Требования

* JDK 17+
* MySQL 8.x (localhost:3306)

## Установка БД

1. В MySQL выполнить: `SOURCE create_db.sql;`
2. При необходимости сменить креды в `src/main/java/org/example/Main.java` (константы `DB_URL` / `DB_USER` / `DB_PASS`).

## Запуск

* Сборка JAR:  
  `mvn -q -DskipTests package`
* Запуск:  
  `java -jar target/string-menu-mysql-1.0.0-shaded.jar`

## Проверка пунктов меню

1 — вывести все таблицы из MySQL (проверка подключения к БД и выбранной схемы)  
2 — создать/проверить таблицу `string_results`  
3 — ввести две строки (не менее 50 символов), сохранить в MySQL и вывести строки в консоль  
4 — посчитать длину ранее введённых строк (`len1`, `len2`), обновить запись в MySQL и вывести результат в консоль  
5 — объединить две строки в одну (`concat_str`), сохранить в MySQL и вывести результат в консоль  
6 — сравнить две строки (лексикографически), сохранить текстовый результат сравнения (`compare_result`) и вывести его в консоль  
7 — экспортировать все записи из `string_results` в файл `string_results.xlsx` и вывести путь к файлу в консоль

> Для пунктов 4–6 используется **последняя сохранённая запись** в таблице `string_results` (по максимальному `id`).

## Примечания

* Каждая строка при вводе для пункта 3 должна быть **не короче 50 символов** — это проверяется в коде.
* Таблица `string_results` создаётся либо через `create_db.sql`, либо через пункт меню 2.
* Файл Excel `string_results.xlsx` создаётся рядом с исполняемым JAR  
  (или в корне проекта при запуске из IDE).
